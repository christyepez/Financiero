# Sprint 00 - Portal Discovery

## Goal

Before implementing any financial module, inspect `PortalCorporativo` and document what can be reused.

## Inputs

```text
https://github.com/christyepez/PortalCorporativo
README.md
AGENTS.md
codex/INSTRUCTIONS.md
backend
frontend/portal-angular
infrastructure/sql
docker-compose.yml
```

## Tasks

1. Inspect backend APIs.
2. Inspect Angular portal shell.
3. Inspect security model.
4. Inspect menu model.
5. Inspect audit API.
6. Inspect notification API.
7. Inspect integration API.
8. Inspect reporting API.
9. Inspect content API.
10. Inspect catalog/configuration APIs.
11. Inspect Docker Compose.
12. Inspect outbox/workers.
13. Generate reuse matrix.
14. Generate do-not-duplicate list.
15. Generate extension plan.

## Exit Criteria

```text
portal-capability-inventory.md exists.
portal-reuse-matrix.md is updated with real paths.
do-not-duplicate-list.md exists.
portal-extension-plan.md exists.
No code has been created yet.
```
