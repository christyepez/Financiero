# Sprint 01 - Portal Integration Foundation

## Goal

Create the adapters and contracts needed to reuse PortalCorporativo APIs.

## Deliverables

- Audit adapter.
- Notification adapter.
- Security extension catalog.
- Menu registration seed.
- Configuration reader contract.
- Content storage contract.
- Integration registry contract.
- Outbox reuse ADR.

## Exit Criteria

- Test event reaches portal audit mock.
- Test notification reaches portal notification mock.
- Finance menu can be registered.
- Finance permissions can be seeded.
