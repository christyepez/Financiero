# SRI Validation Evidence Template

Objective: capture non-sensitive evidence for SRI test/manual validation.

Scope: RIDE representation, SRI test responses and ATS validation when enabled.

Responsible: integration owner and tax reviewer.

Required evidence:
- Environment used.
- Synthetic document identifiers.
- Sanitized request/response summary.
- Validation result.
- Error mapping if rejected.

Decision: accepted / rejected / needs rework.

Approval: reviewer name, date and evidence reference.

Risks: accidental real taxpayer data, real XML, certificates or production URLs.

Blocking items: missing sanitization or production/test ambiguity.
