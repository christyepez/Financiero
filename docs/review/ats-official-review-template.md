# ATS Official Review Template

Objective: validate readiness for future official ATS generation.

Scope: informant, sales, purchases, withholdings, voided documents, establishments, tax summary, catalogs and schema.

Responsible: external Ecuador tax reviewer.

Required evidence:
- Official catalog version.
- Official schema/XSD version.
- Synthetic data validation examples.
- Purchase and voided document coverage.
- Retention code and percentage review.

Decision: approved / rejected / deferred.

Approval: reviewer name, role, date and signature reference.

Risks: incomplete modules, invalid catalogs, invalid schema mapping.

Blocking items: missing official schema, missing purchase module, missing voided document model.
