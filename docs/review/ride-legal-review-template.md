# RIDE Legal Review Template

Objective: validate whether the RIDE layout can move from foundation to official-ready.

Scope: factura, nota de crédito, nota de débito and comprobante de retención.

Responsible: external tax/legal reviewer.

Required evidence:
- Final layout samples using synthetic data.
- Mandatory texts and authorization representation.
- QR/access-key display decision.
- Totals, taxes and issuer data review.

Decision: approved / rejected / deferred.

Approval: reviewer name, role, date and signature reference.

Risks: wrong legal wording, incomplete authorization display, sensitive data exposure.

Blocking items: any critical gap without evidence.
